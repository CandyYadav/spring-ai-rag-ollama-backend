# Spring AI RAG with Ollama and pgvector

This project integrates Spring AI with Ollama and pgvector, providing a robust framework for building applications utilizing retrieval-augmented generation (RAG). This combination enables developers to leverage powerful AI capabilities, enhancing the functionality and performance of their Spring applications.

## Key Features
- **Spring AI Framework**: Seamlessly integrates with the Spring ecosystem, offering a familiar programming model.
- **Ollama Integration**: Connects with Ollama to facilitate advanced AI processing and generate responses based on user queries.
- **pgvector Support**: Utilizes pgvector to optimize vector similarity searches, enabling efficient data retrieval and processing.
